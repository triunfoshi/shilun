# 三期原子规则规范

原子规则是三期策略中心的最小可解释单元。它只负责把已有字段组合成候选标签，不负责直接给买卖结论、排序权重或仓位。

## 规则接口

每条规则至少包含：

| 字段 | 含义 |
| --- | --- |
| `code` | 规则和标签唯一编码 |
| `factor_group` | 所属因子组，可多组 |
| `description` | 规则目的 |
| `required_fields` | 依赖字段清单 |
| `validation_path` | 后续验证报告路径 |
| `evaluator` | 输入 `RuleContext`，输出 `CandidateTag` 或 `None` |

运行期输入：

| 输入 | 含义 |
| --- | --- |
| `ticker` | 股票代码 |
| `snapshot` | 单票分析结果快照 |
| `latest_bar` | 最新日线 bar |
| `metadata` | 股票名称、行业、市场板块及增强上下文 |

运行期输出：

| 输出 | 含义 |
| --- | --- |
| `code` | 候选标签 |
| `factor_group` | 标签因子组 |
| `reason` | 命中原因 |
| `score` | 标签内部观察分，不参与主排序 |
| `validation_path` | 验证报告路径 |
| `source` | 策略来源借鉴 |

## 首批规则口径

### `rps_breakout`

- 来源：Sequoia-X
- 含义：强势相对表现叠加突破胜率
- 当前口径：`structure_score`、`entry_probability`、`p_breakout_success`、市场分、板块强度同时达标
- 验证重点：补横截面 RPS、分市场阶段验证 1/3/5/10 日收益

### `turtle_breakout`

- 来源：Sequoia-X
- 含义：海龟式真实突破观察
- 当前口径：突破风格、突破成功率、温和放量、低分布压力、风险可控
- 验证重点：突破后失败率、回撤、换手和滑点敏感性

### `high_tight_flag`

- 来源：Sequoia-X、Alpha
- 含义：强势整理或高位紧凑形态
- 当前口径：中期趋势强、放量质量好、滞涨和分布压力低
- 验证重点：高位整理后延续概率、假突破率、不同板块表现

### `earnings_surprise`

- 来源：BaoStock 交易技术商城
- 含义：业绩预告或利润/收入增速超预期
- 当前口径：事件字段达标且市场环境不差
- 验证重点：公告后收益窗口、预告类型、增长来源和行业景气度

### `dividend_quality`

- 来源：BaoStock 交易技术商城
- 含义：分红稳定性与股东回报质量
- 当前口径：现金分红收益率、连续分红年数、分红稳定性与基本面质量
- 验证重点：熊市防守、低波动贡献、与趋势类标签组合后的收益变化

## 升级门禁

- `观察`：规则可输出候选标签，但不影响排序。
- `验证中`：有可复现实验脚本和初版报告。
- `保留`：样本内、样本外、分市场阶段均通过最低标准。
- `升主链`：允许进入排序、组合仓位或执行规则，需要引用验证报告。
- `淘汰`：收益无优势、未来函数风险、样本外失效或解释不可复现。
