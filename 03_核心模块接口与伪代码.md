# 第一批核心模块接口定义和伪代码

## 1. 数据导入模块

### `CSVImporter.import_csv`

```python
def import_csv(file_path: str, source: str = "csv") -> ImportResult:
    """
    读取 CSV，完成字段映射与基本校验，返回标准化 DataFrame 和导入统计
    """
```

伪代码：

```text
读取 CSV
校验必须字段是否存在
统一列名为 ticker/date/open/high/low/close/volume/amount
统一 date 格式为 YYYY-MM-DD
调用 BarCleaner.clean_bars
写入 bars_daily
返回导入条数、跳过条数、错误条数
```

## 2. 行情清洗模块

### `BarCleaner.clean_bars`

```python
def clean_bars(df: pd.DataFrame) -> pd.DataFrame:
    """
    去重、排序、校验 OHLC 关系、标记异常数据
    """
```

伪代码：

```text
按 ticker/date 排序
删除重复 ticker/date
删除 open/high/low/close 缺失行
检查 high >= max(open, close, low)
检查 low <= min(open, close, high)
标记停牌行和极端异常量能
返回清洗后的 DataFrame
```

## 6. Pivot 检测

### `PivotDetector.detect`

```python
def detect(df: pd.DataFrame, left_bars: int = 3, right_bars: int = 3) -> list[Pivot]:
    """
    检测 pivot high / pivot low
    """
```

伪代码：

```text
遍历每根 K 线
若当前 high 是左右窗口最高，则记为 high pivot
若当前 low 是左右窗口最低，则记为 low pivot
过滤相邻同类 pivot
过滤幅度不足和 bar 数不足的 pivot
返回标准化 Pivot 列表
```


## 9. 事件识别

### `EventEngine.detect`

```python
def detect(df: pd.DataFrame, zones: list[Zone]) -> list[StructureEvent]:
    """
    识别 breakout / pullback / failed_breakout
    """
```

伪代码：

```text
取最近活跃 zone
若 close 突破 zone_upper 且幅度与量能满足条件，生成 breakout_up
若突破后回踩 zone_upper 且未跌破失效位，生成 pullback_confirm
若突破后短期跌回 zone 内，生成 failed_breakout
按优先级去重事件
返回事件列表
```

## 10. 动作映射

### `ActionMapper.map_actions`

```python
def map_actions(snapshot: StructureSnapshot) -> ActionDecision:
    """
    根据结构状态、事件和风险标签输出未持仓/已持仓建议
    """
```

伪代码：

```text
如果 trend_weekly=up 且事件为 breakout_up:
    未持仓 = watch 或 buy_on_pullback
    已持仓 = hold_above_support
如果事件为 failed_breakout:
    未持仓 = avoid_chasing
    已持仓 = exit_on_invalidation
若 weekly/daily 冲突:
    附加 weekly_daily_conflict
返回动作建议对象
```
